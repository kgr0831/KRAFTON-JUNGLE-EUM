# 작업 완료 시 체크리스트

## 코드 변경 후

### Frontend
1. `npm run lint` - ESLint 검사
2. `npm run build` - 빌드 성공 확인
3. 브라우저에서 기능 테스트

### Backend (Go)
1. `go fmt ./...` - 코드 포맷팅
2. `go vet ./...` - 정적 분석
3. `go build ./...` - 빌드 성공 확인
4. `go test ./...` - 테스트 실행 (있는 경우)

### Python
1. 코드 실행 테스트
2. gRPC 서비스 응답 확인

## Proto 변경 시
1. Go 코드 재생성
2. Python 코드 재생성
3. 양쪽 서버 재시작

## 커밋 전
1. `.env` 파일 커밋 제외 확인
2. 불필요한 로그/디버그 코드 제거
3. 의미 있는 커밋 메시지 작성

## 배포 전
1. 환경 변수 설정 확인
2. Docker 이미지 빌드 테스트
3. 로컬에서 전체 통합 테스트
